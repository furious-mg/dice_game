• Each player will roll the dice, beginning with the first player. The number on the dice will be added to the current score of the player.
  • The player can roll the dice as many times as he wants, there are only two conditions for termination
  • Condition 1: If the player gets the number 1 on the dice, the turn for rolling the dice will be passed to the other player. In this case, the current score will not be added to the main score and the current score will be set to zero
  • Condition 2: If the player playing the turn chooses to hold the score, then the current score of the player will be added to the main score of the player and the current score will be reset to 0
  • The player whose main score reaches 100 first will be declared as the winner
